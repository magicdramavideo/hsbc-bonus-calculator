# HSBC 2026 理專獎金計算系統 - 設計文件

## 設計理念

本應用程式為匯豐銀行理財專員設計，用於快速計算季度獎金。設計遵循 iOS Human Interface Guidelines，提供簡潔、專業的金融工具體驗。

## 螢幕列表

### 1. 主畫面（Home Screen）
- **主要內容**：職級選擇、認列比例輸入、業績目標顯示
- **功能**：
  - 職級選擇器（Picker）：9個職級選項
  - 認列比例輸入框（0-100%）
  - 顯示當前職級的月目標、QTI、折扣後目標
  - 導航至計算畫面的按鈕

### 2. 計算畫面（Calculator Screen）
- **主要內容**：財務指標與非財務指標輸入表格、達成率顯示
- **功能**：
  - 財務指標輸入區：投資手收、保險手收、CA、NNM、Wealth Penetration
  - 非財務指標輸入區：Risk、Quality、Complaint、Client Appointment、NPS
  - 即時計算各項達成率（百分比顯示到小數點後2位）
  - 顯示財務指標總分、非財務指標總分
  - 顯示預估獎金金額
  - 儲存紀錄按鈕

### 3. 歷史紀錄畫面（History Screen）
- **主要內容**：已儲存的計算紀錄列表
- **功能**：
  - 顯示所有儲存的計算紀錄（日期、職級、預估獎金）
  - 點擊紀錄查看詳細資料
  - 刪除紀錄功能

### 4. 紀錄詳情畫面（Record Detail Screen）
- **主要內容**：單筆紀錄的完整資料
- **功能**：
  - 顯示職級、認列比例、所有輸入數據
  - 顯示所有達成率與計算結果
  - 編輯或刪除紀錄

## 主要使用者流程

1. **新增計算**：
   - 使用者開啟應用程式 → 主畫面
   - 選擇職級 → 系統顯示該職級目標與QTI
   - 輸入認列比例 → 系統計算折扣後目標
   - 點擊「開始計算」→ 計算畫面
   - 輸入財務與非財務指標 → 系統即時計算達成率與獎金
   - 點擊「儲存紀錄」→ 紀錄儲存成功提示

2. **查看歷史紀錄**：
   - 使用者點擊「歷史紀錄」標籤 → 歷史紀錄畫面
   - 瀏覽紀錄列表 → 點擊特定紀錄 → 紀錄詳情畫面
   - 查看完整資料或編輯/刪除

3. **編輯紀錄**：
   - 從紀錄詳情畫面 → 點擊「編輯」→ 返回計算畫面（預填資料）
   - 修改數據 → 儲存更新

## 色彩選擇

遵循匯豐銀行品牌色彩，營造專業金融氛圍：

- **主色調（Primary）**：HSBC 紅色 `#DB0011`（用於重要按鈕、強調元素）
- **次要色調（Secondary）**：深灰色 `#333333`（用於文字、圖示）
- **背景色（Background）**：淺灰色 `#F5F5F5`（淺色模式）、深灰色 `#1C1C1E`（深色模式）
- **卡片背景（Card）**：白色 `#FFFFFF`（淺色模式）、深灰色 `#2C2C2E`（深色模式）
- **成功色（Success）**：綠色 `#34C759`（達成率 ≥ 100%）
- **警告色（Warning）**：橙色 `#FF9500`（達成率 70-99%）
- **錯誤色（Error）**：紅色 `#FF3B30`（達成率 < 70%）

## 版面配置

- **間距系統**：8pt 網格（8、12、16、24、32）
- **圓角**：按鈕 8pt、卡片 12pt、輸入框 8pt
- **字體大小**：
  - 標題：28pt（粗體）
  - 副標題：20pt（粗體）
  - 正文：16pt
  - 說明文字：14pt
  - 小字：12pt
- **觸控目標**：所有可點擊元素最小 44pt
- **安全區域**：所有內容尊重 iOS 安全區域（刘海、底部指示器）

## 導航結構

- **底部標籤列（Tab Bar）**：
  1. 首頁（Home）- 職級選擇與計算
  2. 歷史（History）- 紀錄列表
- **堆疊導航（Stack Navigation）**：
  - 主畫面 → 計算畫面
  - 歷史紀錄畫面 → 紀錄詳情畫面

## 圖示風格

- 使用 SF Symbols 風格圖示（iOS）
- 所有圖示使用填充（filled）樣式，保持一致性
- 標籤列圖示大小：28pt
- 按鈕圖示大小：20pt

## 資料儲存

- 使用 AsyncStorage 進行本地資料儲存
- 不需要雲端同步或使用者帳號
- 儲存內容：計算紀錄（職級、認列比例、各項指標、計算結果、時間戳記）
